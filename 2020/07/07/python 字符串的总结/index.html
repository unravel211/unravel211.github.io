python字符串的总结

python字符串的总结

1.字符串的字符:

1.转义字符: 在指定符号前面加’\‘来表示一些特殊功能和意义的字符

2.普通字符: 除了转义字符都是普通字符

3.阻止转义(r语法): 在字符串最前面 + r/R,会使得这个字符串中所有转义字符都无效

  • -> 使用1:windows电脑写路径可使用

  • -> 使用2:正则表达式中,\会与\w\d这些符号冲突,所以使用正则表达式的时候都会在前面加上r,做阻止转义

2.字符编码:

计算机存储数据时只能存数字,存的是数字补码

1)ASCII码表:用1个字节带队字符进行编码(码值:0~127)

a.数字在字母前面(字符0~字符9的编码值连续递增)
b.大写字母在小写字母前面(字母之间是连续递增)
c.大小写字母之间不是连续的,有其他符号

2)Unicode编码表

Unicode编码表是对ASCII表的扩展(Unicode编码表中包含了ASCII表中的所有)
Unicode编码表中包含世界上所有国家的所有语言
中文范围: 4e00~9fa5
